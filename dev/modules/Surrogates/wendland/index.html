Wendland Surrogate

The Wendland surrogate is a compact surrogate: it allocates much less memory then other surrogates. The coefficients are found using an iterative solver.

$f = x -> exp(-x^2)$

using Surrogates
using Plots
n = 40
lower_bound = 0.0
upper_bound = 1.0
f = x -> exp(-x^2)
x = sample(n,lower_bound,upper_bound,SobolSample())
y = f.(x)

We choose to sample f in 30 points between 5 to 25 using sample function. The sampling points are chosen using a Sobol sequence, this can be done by passing SobolSample() to the sample function.

Building Surrogate

The choice of the right parameter is especially important here: a slight change in Ïµ would produce a totally different fit. Try it yourself with this function!

my_eps = 0.5
wend = Wendland(x,y,lower_bound,upper_bound,eps=my_eps)
